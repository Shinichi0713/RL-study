__練習問題__
以下のように5×5の迷路を使います。

Q学習またはSARSAを使って「スタートからゴールまで最短経路を学習するプログラム」を作成してください。
```
maze = [
    ['S', '.', '.', '#', '.'],
    ['.', '#', '.', '#', '.'],
    ['.', '#', '.', '.', '.'],
    ['.', '.', '#', '#', '.'],
    ['#', '.', '.', 'G', '.']
]
S: スタート
G: ゴール
#: 壁（通れない）
.: 通れるマス
```

## 構成
1. 環境
迷路、リワードする、状態を返す、壁は通さない
2. エージェント
行動選択する、行動価値を更新する
3. 訓練コード
エージェントを訓練する
